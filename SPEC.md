# 学习助手 (StudyAssistant) - 规格文档

## 1. 项目概述

- **项目名称**: 学习助手 (StudyAssistant)
- **类型**: Android 原生应用
- **核心功能**: 番茄钟计时器，计时期间锁定设备，支持提前解锁挑战
- **离线支持**: 所有功能完全离线可用，无需网络连接

## 2. 技术栈

- **语言**: Kotlin 1.9.x
- **最低 SDK**: API 26 (Android 8.0)
- **目标 SDK**: API 34 (Android 14)
- **UI**: Jetpack Compose + Material Design 3
- **架构**: MVVM + Clean Architecture
- **DI**: Hilt（待集成）
- **异步**: Kotlin Coroutines + Flow
- **本地存储**: DataStore Preferences（待集成）
- **构建**: Gradle 8.4 + AGP 8.2.2

## 3. 功能列表

### F1: 计时器设置
- 预设时间选项: 25分钟 / 45分钟 / 60分钟 / 90分钟
- **自定义时间设置**（✨已更新）:
  - 小时输入：0~24
  - 分钟输入：0~59
  - 秒输入：0~59
  - 最长支持 **24小时**
- 显示剩余时间（时:分:秒格式）

### F2: 专注锁定（防锁机保护）
- 点击「开始专注」后启动倒计时
- 计时期间激活 Android Device Policy Manager (DPM) 策略，限制解锁屏幕
- 计时期间界面显示倒计时，不可退出或返回
- **安全保护措施**（✨新增）:
  - 自动超时解锁（默认30分钟后自动解锁）
  - 应用启动时检查异常退出状态并自动解锁
  - 紧急解锁：专注界面连续点击5次可强制解锁
  - SharedPreferences 持久化锁定状态，防止崩溃后锁机

### F3: 提前解锁挑战
- 点击「提前解锁」进入数学挑战模式
- **年级范围自定义**（✨已更新）:
  - 起始年级下拉选择
  - 结束年级下拉选择
  - 自动确保结束年级 ≥ 起始年级
- 题目从年级范围内随机抽取
- 连续答对指定数量题目（3/5/10题）后自动解锁
- 答错题目显示错误提示，可跳过换题
- 完全离线运行，题目本地生成

### F4: 计时结束
- 倒计时归零后自动解除锁定
- 显示「专注结束」提示
- 可重新设置新一轮计时

### F5: 全程静音
- 所有操作不触发任何声音（系统静音 + AudioManager 控制）
- 倒计时结束不播放提示音

### F6: 离线可用
- 所有功能完全离线运行
- 无需网络连接即可使用全部功能
- 数学题目本地生成算法，无需联网获取

## 5. UI 设计

- **配色**: 深色主题（护眼），主色 #4CAF50（绿色），强调色 #FFC107（琥珀色）
- **布局**: 单屏设计，中央圆形倒计时 + 底部设置选项
- **字体**: 大字号倒计时数字，清晰易读
- **交互**: 简洁按钮，开始/取消专注、提前解锁

## 6. 关键实现

### 锁定机制
使用 Android Device Administration API:
- 注册 DeviceAdminReceiver
- 用户首次使用时授权设备管理员权限
- 调用 `devicePolicyManager.setKeyguardDisabled(admin, true)` 锁屏
- 调用 `devicePolicyManager.setKeyguardDisabled(admin, false)` 解锁

### 静音机制
```kotlin
audioManager.ringerMode = RINGER_MODE_SILENT
```

### 数学题目生成
本地算法生成，按年级划分难度：
- 加减乘除四则运算
- 平方、平方根、幂运算、阶乘
- 一元一次方程求解
- 百分比、几何计算
- 负数、对数、数列
